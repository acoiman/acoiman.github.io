Retrieving, Analyzing and Visualizing georeferenced data

Earthquakes near Fiji since 1964

Apr 25, 2020 1 min read

This post contains three jupyter notebooks that will show you how to map earthquakes from a database using standard Python and Folium libraries. The database was filled out from a CSV file obtained from the Rdatasets .

  1. reading_dataset contains the code to read through the Rdatasers and look for dataset links containing the terms latitude and longitude.

  2. db_earthquakes creates a database from the selected dataset and computes some spatial statistics.

  3. map_earthquakes takes the database data and creates a web map using the Folium package.

Repository link: https://github.com/acoiman/mapping_earthquakes
