# Abraham Coiman — Personal Website

> **Live site → [https://acoiman.github.io/](https://acoiman.github.io/)**

Personal portfolio of Abraham Coiman

---

## About this repository

This repository hosts the **compiled, production-ready output** of the portfolio website. The files here are generated automatically by [Vite](https://vitejs.dev/) and are not meant to be edited directly.

The source code (React components, assets, configuration) is stored in a separate private repository.

---

## Sections

- **Home** — greeting and profile photo
- **About Me** — summary, skills, and professional experience
- **Publications** — carousel of research publications
- **Maps** — carousel of interactive geospatial maps
